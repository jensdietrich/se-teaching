## Justification

Jackson is the most widely used JSON library in the Java ecosystem (top-ranked on Maven Central). Used here to serialise request bodies sent to the Ollama REST API and to deserialise the JSON responses. FasterXML is a reputable provider with active maintenance and a large contributor base.
